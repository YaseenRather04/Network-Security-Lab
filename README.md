# Network Security Lab

This repo contains the 10 Experiments we did in 4th sem Network Security Lab

## 🧪 List of Experiments

* **Experiment 1**
  * **Aim:** Basic network configuration on Cisco device.

* **Experiment 2**
  * **Aim:** To implement VLAN on switch & configure inter-VLAN communication.

* **Experiment 3**
  * **Aim:** To implement Inter-VLAN communication using Multilayer Switch.

* **Experiment 4**
  * **Aim:** To implement A.C.L to restrict traffic & specific host.

* **Experiment 5**
  * **Aim:** To configure DHCP to dynamically assign IP.

* **Experiment 6**
  * **Aim:** Configure NAT to allow internal device to accept external network.

* **Experiment 7**
  * **Aim:** Installing NS2 in Ubuntu & creating a basic network.

* **Experiment 8**
  * **Aim:** Setup and simulate a basic network topology.

* **Experiment 9**
  * **Aim:** Simulate common attack like DOS in NS2.

* **Experiment 10**
  * **Aim:** Simulate & analyze the performance of Network Security protocol.
